# 页面组成

[[toc]]

## Layout
aftershock 页面的Layout主要分Header、Main和Footer作为项目的主体部分，在所有页面通用，当然是被划分为不同的组件进行渲染。

### Header
- Logo 点击跳转到首页

![aftershock](/screenshots/ScreenShot_2025-11-25_163541_877.png "aftershock")

- Menu (菜单栏) 菜单组件，在primics树形配置 [数据源]()

![aftershock](/screenshots/9c0397bd-854b-495b-93d2-5a23a757dbed.png "aftershock")

- Search 搜索组件，可以用来搜索商品链接，点击之后跳转到对应的商品界面

![aftershock](/screenshots/ScreenShot_2025-11-25_164420_805.png "aftershock")

- 地区切换 点击之后切换不同地区的网站，目前只支持澳洲、新加坡

![aftershock](/screenshots/2025-11-25_164556_912.png "aftershock")

- 联系我们 在点击之后，会从右侧弹出可选的联系方式与 aftershock 的相关人员联系，并及时获得反馈。

![aftershock](/screenshots/2025-11-25_164759_510.png "aftershock")

- 用户信息

    - 用户未登录状态会提醒用户登录或者注册

    ![aftershock](/screenshots/ScreenShot_2025-11-25_165046_924.png "aftershock")

    - 登录状态会弹出用户个人信息选项和aftershock的联系方式

    ![aftershock](/screenshots/ScreenShot_2025-11-25_165605_550.png "aftershock")

- 购物车 显示用户的购物车商品和推荐的商品

![aftershock](/screenshots/ScreenShot_2025-11-25_171404_546.png "aftershock")

- 横幅 一般显示商城的活动，以及标语等，点击跳转到对应的界面

![aftershock](/screenshots/20251125171759_32_41.png "aftershock")
    

## Footer

Footer 组件展示了当前我们的各类服务性页面，其中有产品组、我们的公司信息、帮助文档，以及联系方式和媒体账号。

![aftershock](/screenshots/ScreenShot_2025-11-28_173914_746.png "aftershock")
