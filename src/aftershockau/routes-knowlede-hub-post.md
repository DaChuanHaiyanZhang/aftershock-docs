# Knowlede Hub Post

该页面主要用来显示具体的知识内容，包含各类图文知识的说明等。

![aftershock](/screenshots/ScreenShot_2025-11-27_181106_433.png "aftershock")


## 侧边栏

侧边栏表示知识的具体分组，在点击之后能够快速跳转。

![aftershock](/screenshots/ScreenShot_2025-11-27_181225_238.png "aftershock")

## 面包屑导航

通过点击面包屑，可以快速回到上个分组或者上个页面

![aftershock](/screenshots/ScreenShot_2025-11-27_181420_862.png "aftershock")
