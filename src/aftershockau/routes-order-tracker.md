# Order tracker

订单追踪界面

该界面主要用来查看订单状态。

用户在输入对应的订单号之后，会查询到订单的状态。

![aftershock](/screenshots/ScreenShot_2025-12-01_141404_647.png "aftershock")


>[!WARNING]
>登录状态下，用户只需要输入订单号查询，否则则需要订单号和邮箱号
